# AWS Certified Data Engineer Associate (DEA-C01)

A comprehensive course for cloud engineers preparing for the AWS Certified Data Engineer Associate exam, covering all AWS-native data services extensively.

---

## How to use this course

Use the **left sidebar** to navigate between sections. Each section covers one AWS service or domain in depth with:

- Conceptual explanation and architecture diagrams
- AWS service integrations (what connects to what)
- Security — IAM, KMS, VPC, CloudTrail
- Performance tuning with specific config values
- Important CloudWatch metrics to monitor
- Hands-on code examples (boto3 / AWS CLI / SQL)
- Common FAQs and exam misconceptions
- Exam focus areas mapped to DEA-C01 domains
- Blog and reference implementations
